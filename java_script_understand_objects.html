Objects in Javascript

Create a object name "player" and add attributes to it

var player = {firstName:"Peter", lastNmae:"England", age:32, city:"Atlanta"};

Print each object by using object.attribute

"player.firstname" will return

"player.city" will return

Add a method (function) to object to print full name

var player = { firstName:"Peter", lastNmae:"England", printName:function() { this.firstName + " " + this.lastName }};

Now call method printName to print full name like player.printName()

Now call method printWelcomeMessage to print welcome message like player.printName()
